# 📓 Journal des Apprentissages (Learnings)

Ce fichier est la mémoire vive de Cortex. Il recense les points de friction, les erreurs récurrentes et les préférences de Romain pour permettre aux agents de s'auto-améliorer.

## Comment l'utiliser ?

1. **Identifier** : Un agent a fait une erreur ou n'a pas respecté une préférence.
2. **Noter** : Ajouter une entrée dans la section "Inbox" ci-dessous.
3. **Traiter** : L'agent `knowledge-curator` lira cette inbox pour mettre à jour les agents.

---

## 📥 Inbox (À traiter)

- [ ] **Exemple** : Les agents ont tendance à utiliser des IDs génériques (`btn-1`). Préférence de Romain : utiliser des IDs sémantiques (`submit-recipe-button`).
- [ ] **Réseau** : Le `git push` en ligne de commande est bloqué. Toujours utiliser l'outil `push_files` via API GitHub pour synchroniser le repo Cortex.
